    new RegisterApiHelper.RegisterCallback() {
        public void onRegisterSuccess(AuthResponse response) {
            // ✅ Thành công! Token: response.getToken()
        }
        public void onRegisterError(String error) {
            // ❌ Thất bại! Lỗi: error
        }
    }
);

┌──────────────────────────────────────────────────────────────────────────┐
│  ⚠️ LƯU Ý QUAN TRỌNG                                                     │
└──────────────────────────────────────────────────────────────────────────┘

  1. SERVER URL
     • Emulator: http://10.0.2.2:8080/
     • Real device: http://YOUR_IP:8080/
     • Đổi BASE_URL trong RetrofitClient.java nếu cần

  2. TOKEN MANAGEMENT
     • Lưu token vào SharedPreferences sau khi đăng ký
     • Token sẽ được dùng cho các API call khác
     • Thêm token vào header: "Authorization: Bearer <token>"

  3. ERROR HANDLING
     • Network error → Check internet connection
     • Username exists → Use different username
     • Email exists → Use different email
     • Invalid data → Check validation

  4. VALIDATION
     • Username: min 3 characters
     • Email: valid format
     • Password: min 6 characters
     • Confirm password: must match
